# Medical Device Market Analysis

## 📌 Overview
This project analyzes medical device market data to identify trends, product performance, and customer insights.

## 🛠 Tools Used
- Python (Pandas)
- SQL (MySQL)
- Excel
- Power BI

## 🚀 Project Workflow
1. Data Cleaning using Python
2. Data Storage in MySQL
3. SQL Analysis (Joins, Aggregations, Window Functions)
4. Dashboard Creation (Power BI)

## 📊 Key Insights
- Identified top-performing products by revenue
- Analyzed region-wise market trends
- Evaluated customer satisfaction scores

## 📁 Folder Structure
- data/ → Raw & cleaned data
- scripts/ → Python scripts
- sql/ → SQL queries
- dashboard/ → Power BI file

## 💡 Skills Demonstrated
- Data Cleaning
- SQL Analysis
- Data Visualization
- Business Insights

## 🎯 Goal
To transition from Pharmacovigilance to Data Analytics by working on real-world healthcare datasets.
